Movie Entity Search Engine

End-to-end data science project that goes from raw tabular wrangling to a production-style movie search engine.

Built using the MovieLens 25M dataset from GroupLens Research.

The goal is to practice real-world ML engineering skills:
	•	data wrangling
	•	multi-table joins
	•	datetime features
	•	sklearn pipelines
	•	imputation
	•	grid search
	•	text similarity search
	•	API deployment

Final result:
GET /search?q=dark psychological thriller
returns ranked movies.
